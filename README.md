#### 一个简单的小游戏

找相同色块的连连看

简单，节奏紧凑，容易上瘾，但不够耐玩


#### 实现如下功能

- 每关随机生成色块，增加行数

- 剩余时间不多的时候在色块上显示数字，解决颜色太相同的问题
